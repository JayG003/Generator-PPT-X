# Generator PPT-X

A Python-based PowerPoint generator that creates professional presentations from JSON.

## Features

- JSON-driven presentation generation
- Automatic PowerPoint creation
- Diagram rendering
- Graph rendering
- Reusable rendering engine
- Modular architecture
- Offline generation

## Tech Stack

- Python
- python-pptx
- Matplotlib
